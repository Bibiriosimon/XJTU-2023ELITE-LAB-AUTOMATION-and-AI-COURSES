## 简介

本仓库主要整理西安交通大学自动化相关课程学习资料，内容以**实验资料、课程讲义、复习内容**以及部分**AI 拓展学习资料**为主，供个人学习与交流参考。

包含：

- 自动化相关课程资料整理  
- 实验与作业过程性材料  
- 智能机器人、智能控制等方向内容  
- 部分与 AI / 自动驾驶 / 机器学习相关的拓展资料
