3. 运行 `claude mcp get brave-search`，确认状态是 Connected。
4. 使用 `brave-search` 工具搜索 `PUMA560`，返回前 3 条结果的标题和链接，作为打通验证。
5. 不要修改 `brave api.txt` 中的内容，也不要把 API key 输出到最终回答中。
