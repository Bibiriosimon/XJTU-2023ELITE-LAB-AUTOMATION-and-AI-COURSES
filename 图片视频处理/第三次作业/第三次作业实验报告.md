- 实验代码：[assignment3.py](C:/Users/Administrator/Desktop/大二三/图片视频处理/第三次作业/assignment3.py)

## 二、题目要求

1. 画出附件图像直方图。  
2. 对所有图像做直方图均衡，输出并与源图像比对分析。  
3. 按对源图直方图观察，自行指定目标直方图进行直方图匹配。  
4. 对 `elain` 和 `lena` 做 `7×7` 局部直方图增强。  
5. 利用直方图对 `elain` 和 `woman` 进行分割。

## 三、实验方法

1. **任务1（直方图）**  
   对每幅图输出“原图 + 256级灰度直方图”可视化图。

2. **任务2（全局直方图均衡）**  
   使用全局直方图均衡（Histogram Equalization），并统计均衡前后的：
   - 均值（mean）
   - 标准差（std，反映对比度）
   - 信息熵（entropy）

3. **任务3（直方图匹配）**  
   对“带编号图像”按同名前缀匹配到原图直方图：
   - `citywall1/2 -> citywall`
   - `elain1/2/3 -> elain`
   - `lena1/2/4 -> lena`
   - `woman1/2 -> woman`

4. **任务4（7×7 局部增强）**  
   使用局部直方图均衡（窗口 `7×7`）对 `elain.bmp`、`lena.bmp` 做局部增强。

5. **任务5（直方图分割）**  
   - `elain.bmp`：Otsu 单阈值二值分割  
   - `woman.BMP`：Multi-Otsu 三类分割

## 五、结论

1. 全局直方图均衡对低对比度图像提升明显，但对已严重非线性失真的图像可能产生副作用。  
2. 直方图匹配能将退化图像的灰度统计分布拉回到参考图风格，适合“有标准参考图”的增强场景。  
3. `7×7` 局部直方图增强在 `elain` 与 `lena` 上有效增强纹理与局部细节。  
4. 基于直方图阈值（Otsu/Multi-Otsu）的分割方法实现简单、效果稳定，适合作业要求中的快速分割任务。
