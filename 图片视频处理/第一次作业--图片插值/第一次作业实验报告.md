- 实验名称：BMP格式解析与灰度图像处理
- 实验环境：Python 3.x，`numpy`，`Pillow`，`matplotlib`
- 实验数据：`7.bmp`、`lena.bmp`（512×512）、`elain1.bmp`（512×512）

## 二、实验目的

1. 理解 BMP 文件结构与关键头部字段。
2. 掌握灰度级量化（8bit 到 1bit）对图像视觉效果的影响。
3. 计算图像统计量（均值、方差）并理解其意义。
4. 对比近邻、双线性、双三次三种插值放大效果。
5. 实现几何变换（水平 shear、旋转）并观察插值方法在变换后放大中的表现。

## 三、实验内容与方法

1. 解析 `7.bmp` 文件头（14字节）和 DIB 信息头（40字节）。
2. 将 `lena.bmp` 灰度图进行 8/7/6/5/4/3/2/1 bit 量化并保存可视化结果。
3. 使用 `numpy` 计算 `lena.bmp` 灰度均值与方差。
4. 使用 `Pillow` 的 `NEAREST/BILINEAR/BICUBIC` 将 `lena.bmp` 放大到 `2048×2048`。
5. 对 `lena.bmp` 和 `elain1.bmp` 进行：
   - 水平剪切：`k=1.5`
   - 旋转：`30°`
   - 再分别使用三种插值统一缩放到 `2048×2048`

## 四、实验结果

### 1. BMP 格式解析结果（以 7.bmp 为例）

- 文件标识：`BM`
- 文件大小：`1134 bytes`
- 像素数据偏移：`1078 bytes`
- DIB头大小：`40 bytes`
- 宽×高：`7 × 7`
- 位深：`8 bit`
- 压缩方式：`0`（BI_RGB，无压缩）
- 图像数据大小：`56 bytes`

结果说明：
- `pixel_offset = 1078 = 14 + 40 + 1024`，说明该图像包含 256 色调色板（8bit 常见结构）。
- 像素数据按行 4 字节对齐，`7` 列在 8bit 下每行需补齐到 `8 bytes`，`8×7=56 bytes`，与头部字段一致。

对应文件：[task1_bmp_report.txt](C:/Users/Administrator/Desktop/大二三/图片视频处理/第一次作业/outputs/task1_bmp_report.txt)

### 2. 灰度级逐级递减（8bit 到 1bit）

- 已输出 8 张量化图与总览图。
- 结果路径：
  - [task2_gray_levels_overview.png](C:/Users/Administrator/Desktop/大二三/图片视频处理/第一次作业/outputs/task2_gray_levels_overview.png)
  - `outputs/task2_gray_levels/`

现象分析：
- 从 8bit 向 4bit 递减时，细节开始损失，边缘和阴影区域出现较明显“台阶感”。
- 到 2bit、1bit 时，层次急剧减少，图像主要保留轮廓信息，纹理细节显著丢失。
- 1bit 图像仅保留黑白两级，对应强烈二值化效果。

### 4. 三种插值放大到 2048×2048

结果路径：[outputs/task4_zoom](C:/Users/Administrator/Desktop/大二三/图片视频处理/第一次作业/outputs/task4_zoom)

方法对比：
- 近邻插值（nearest）：速度快，但块状感明显，锯齿突出。
- 双线性插值（bilinear）：平滑性提升，锯齿减轻，但边缘略模糊。
- 双三次插值（bicubic）：边缘与细节保留更好，主观观感最自然。

### 5. 几何变换 + 三插值放大

变换参数：
- 水平 shear：`k=1.5`
- 旋转角度：`30°`

中间变换尺寸（以原图 512×512 为基准）：
- shear 后：`1279×512`
- rotate 后：`700×700`

结果路径：[outputs/task5_transform_zoom](C:/Users/Administrator/Desktop/大二三/图片视频处理/第一次作业/outputs/task5_transform_zoom)

现象分析：
- 在 shear 和 rotate 后，图像存在重采样需求，插值方法差异更加明显。
- 近邻法在倾斜边缘上更易出现锯齿和断裂感。
- 双线性法能缓解锯齿，但细节略软。
- 双三次法通常在边缘连续性与细节清晰度之间取得更优平衡。

## 五、结论

1. BMP 文件结构可通过头部字段直接验证图像位深、尺寸、调色板及数据布局。
2. 灰度量化位数越低，图像层次越少，视觉上会出现明显伪轮廓与细节损失。
3. 均值与方差能有效反映图像整体亮度与灰度离散程度。
4. 对于本实验放大与变换任务，双三次插值整体视觉质量最好，近邻法质量最差但计算最简单。
