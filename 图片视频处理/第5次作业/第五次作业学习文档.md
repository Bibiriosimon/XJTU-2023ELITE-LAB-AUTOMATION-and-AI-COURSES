# 第五次作业学习文档

## 一、题目在做什么

这次作业的核心不是再做空域卷积，而是把图像变换到频域之后再设计滤波器。
题目分成三部分：
1. 频域低通：对 `test1`、`test2` 用 Butterworth 和 Gaussian 低通滤波器做平滑。
2. 频域高通：对 `test3`、`test4` 用 Butterworth 和 Gaussian 高通滤波器增强边缘。
3. 其他高通：对 `test3`、`test4` 再做 Laplacian 和 Unsharp，并和频域高通结果对比。

## 二、为什么要看功率谱比

频域滤波器本质上是在保留或抑制某些频率分量。
为了量化“保留了多少频域能量”，这里定义功率谱比：

`power_ratio = sum(|G(u,v)|^2) / sum(|F(u,v)|^2)`

其中 `F` 是原图频谱，`G` 是滤波后频谱。
- 低通滤波：功率谱比越接近 1，说明保留的总能量越多；越小则说明平滑更强。
- 高通滤波：功率谱比通常比较小，因为高频本来只占总能量的一部分。

## 三、参数怎么选

这次我给每张图选择了一个相对合适的截止半径：
- `test1.pgm`：低通半径 `D0=25`
- `test2.tif`：低通半径 `D0=50`
- `test3_corrupt.pgm`：高通半径 `D0=10`
- `test4.tif`：高通半径 `D0=30`

原则是：
- 半径太小：低通会过模糊，高通会过于激进，只剩很少边缘。
- 半径太大：低通效果不明显，高通也难以突出边缘。

## 五、每种方法怎么写优缺点

### 1. Butterworth 低通
- 优点：截止特性比 Gaussian 更明显，平滑作用较强。
- 缺点：在截止区域过渡不如 Gaussian 平滑，可能有轻微振铃。

### 2. Gaussian 低通
- 优点：频域响应平滑，结果自然，边缘过渡柔和。
- 缺点：抑制高频时没有那么“硬”，某些噪声残留会更多。

### 3. Butterworth 高频
- 优点：边缘增强更明显，高频保留更直接。
- 缺点：对噪声也更敏感。

### 4. Gaussian 高频
- 优点：增强较平滑，不容易产生明显伪影。
- 缺点：边缘强化程度通常比 Butterworth 略弱。

### 5. Laplacian
- 优点：对灰度突变非常敏感，轮廓突出。
- 缺点：噪声会被同步放大。

### 6. Unsharp
- 优点：更适合做锐化增强，保留原图外观。
- 缺点：不是纯边缘图，参数不当会过锐。

## 六、空域与频域的关系，报告里该怎么写

题目最后要求讨论空域和频域滤波的关系，这里可以这样写：

1. 理论上，空域卷积等价于频域乘法。
2. 如果频域滤波器 `H(u,v)` 恰好是某个空域冲激响应 `h(x,y)` 的傅里叶变换，那么两者是等效的。
3. 但在实际离散图像处理中，它们往往不会完全一致，原因包括：
- 图像尺寸有限；
- 边界处理方式不同；
- 空域滤波器通常是有限模板，而频域滤波器常是理想化连续函数采样；
- 参数之间不一定一一对应。
4. 因此，空域低通/高通与频域低通/高通在趋势上是一致的，但数值和细节结果未必完全相同。
