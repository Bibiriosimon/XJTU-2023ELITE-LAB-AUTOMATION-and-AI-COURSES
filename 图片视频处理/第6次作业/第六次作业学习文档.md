# 第六次作业学习文档

## 一、题目在做什么

这次作业的核心是“图像退化与恢复”。
前两问是先人为加入噪声，再用不同滤波器恢复；第三问是先加入运动模糊，再加噪声，然后用逆滤波和维纳类方法做复原。

## 二、任务1：高斯噪声恢复

这一步给 Lena 加入高斯噪声，均值设为 0，方差设为 20。
然后分别用了：
- 算术均值滤波
- 中值滤波
- 高斯滤波
- 自适应维纳滤波

为什么这些方法能用：
- 高斯噪声是连续型随机噪声，均值滤波和高斯滤波对它最自然。
- 中值滤波更适合椒盐噪声，所以在高斯噪声场景下不一定最好。
- 维纳滤波会根据局部统计特性调整恢复强度，通常能更平衡噪声抑制和细节保留。

当前实验里，噪声图 PSNR 为 `35.133` dB。
- Arithmetic Mean 3x3: PSNR = `31.940` dB
- Median 3x3: PSNR = `33.058` dB
- Gaussian 5x5: PSNR = `31.696` dB
- Adaptive Wiener 5x5: PSNR = `32.894` dB

写报告时可以说：对于高斯噪声，均值类和平滑类滤波器更合适，其中维纳滤波通常在去噪与保细节之间更平衡。

## 三、任务2：椒盐噪声恢复

这一步给 Lena 加入椒盐噪声，椒和盐的密度都为 0.1。
然后分别用了：
- 算术均值滤波
- 中值滤波
- 反谐波均值滤波（Q=1.5）
- 反谐波均值滤波（Q=-1.5）

反谐波的关键理解：
- `Q > 0` 时，更擅长去除椒噪声（黑点）。
- `Q < 0` 时，更擅长去除盐噪声（白点）。
因为公式中高次或负次幂会对大像素值或小像素值更敏感。

当前实验里，椒盐噪声图 PSNR 为 `12.149` dB。
- Arithmetic Mean 3x3: PSNR = `20.250` dB
- Median 3x3: PSNR = `28.645` dB
- Contra-harmonic Q=1.5: PSNR = `9.455` dB
- Contra-harmonic Q=-1.5: PSNR = `9.011` dB

写报告时可以强调：中值滤波通常是椒盐噪声恢复中最稳妥的选择；反谐波滤波的优势在于可以有针对性地抑制椒噪声或盐噪声。

## 四、任务3：运动模糊 + 维纳恢复

这一步是最核心的。流程是：
1. 构造 45 度方向的运动模糊 PSF。
2. 用它对 Lena 做模糊。
3. 在模糊图中再加入均值 0、方差 10 的高斯噪声。
4. 分别用逆滤波、维纳滤波、约束最小二乘类恢复。

它们的区别：
- 逆滤波：最直接，但对噪声非常敏感。
- 维纳滤波：显式考虑噪声与信号的折中，通常更稳定。
- 约束最小二乘（CLS 风格）：加入平滑约束，对噪声放大更克制。

- Blurred: PSNR = `22.615` dB
- Blurred + Gaussian Noise: PSNR = `22.495` dB
- Inverse Filter: PSNR = `7.904` dB
- Wiener Filter: PSNR = `20.920` dB
- CLS Filter: PSNR = `16.714` dB

报告里可以这样总结：
- 逆滤波在知道模糊函数时理论最直接，但一旦有噪声就很容易失稳。
- 维纳滤波更适合带噪模糊图像恢复。
- CLS 恢复更强调平滑约束，通常视觉上更稳定，但可能牺牲部分锐利边缘。

## 五、这次实验里每种滤波器怎么理解

### 1. 算术均值滤波
- 优点：简单，对高斯噪声有一定抑制作用。
- 缺点：容易模糊边缘，对椒盐噪声不够鲁棒。

### 2. 中值滤波
- 优点：对椒盐噪声效果好，能较好保边缘。
- 缺点：对高斯噪声不一定最优。

### 3. 高斯滤波
- 优点：适合抑制高斯噪声，输出自然。
- 缺点：会带来一定模糊。

### 4. 反谐波均值滤波
- 优点：可以针对性去除椒噪声或盐噪声。
- 缺点：参数 Q 选错时效果会变差。

### 5. 维纳滤波
- 优点：考虑噪声和模糊，是恢复问题里很经典的方法。
- 缺点：依赖噪声估计和模型假设。
